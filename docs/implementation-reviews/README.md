# Implementation Reviews

This directory contains point-in-time architecture integration briefs produced for individual implementation slices. They record the ownership analysis, interaction model, and acceptance criteria that were current at the time each slice was designed or closed.

These documents are historical records, not current specifications. Where they conflict with `CONTEXT.md` or accepted ADRs in `docs/adr/`, the canonical documents govern. In particular, briefs predating the ADR-0004 correction (2026-05-29) may assert Default Scope auto-creation per workspace as a live product behaviour; that behaviour is superseded — there is no Default Scope as a product concept.
